Cinematic Look LUT Library by Flamacore!

How-to:
1-Import Image Effects pack from the Standard Assets package
2-Add a Color Correction(3D Lookup Texture) script from Image Effects to your camera
3-Drag & drop any LUT you wish to the texture selector of the script
4-Convert & apply.

Example scene is provided so that you can inspect the impact of each LUT on a certain color. In order to to run the example scene, again you need Image Effects pack from the Standard Assets package 

Done :)
